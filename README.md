## Instalação

 - Abra o terminal do projecto
 - `npm install` para instalar os pacotes necessários
 - `npm run build` para rodar o webpack
 - `npm start` para rodar o projecto na `localhost:9000`
 - `ctrl + c` para parar o servidor
